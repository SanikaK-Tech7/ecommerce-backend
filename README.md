# E-Commerce Backend

A RESTful backend application built with Java Spring Boot and MySQL.

## Technologies Used
- Java 21
- Spring Boot 4.0.7
- MySQL
- Spring Data JPA
- Maven

## Features
- Add products
- View all products
- Delete products

## API Endpoints
| Method | URL | Description |
|--------|-----|-------------|
| GET | /api/products | Get all products |
| POST | /api/products | Add a product |
| DELETE | /api/products/{id} | Delete a product |
